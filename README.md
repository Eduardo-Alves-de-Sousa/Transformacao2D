# Transformacao2D
- Implementação de programas em C para mostrar o resultado de transformações ou composição de transformações em 2D
- Utilizando OpenGL
## Comandos para rodar os códigos:
- g++ <arquivo.c> -o firstOpenGlApp -lglut -lGLU -lGL
- ./firstOpenGlApp
